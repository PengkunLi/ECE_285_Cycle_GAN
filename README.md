# ECE_285_Cycle_GAN
================

This is project of Style Transfer with Neural Style Transfer and Cycle-GAN developed by Jixuan Liu, Yuhao Tian, Tuoyi Zhao and Yufei Li.

## Requirements:
================

Install package 'easydict' as follow:
        $ pip install --user easydict
